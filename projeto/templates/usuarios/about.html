Sobre o ReqM4IoT

O que é ReqM4IoT?

ReqM4IoT é uma ferramenta online desenvolvida para apoiar desenvolvedores (principalmente aqueles que estejam desenvolvendo sistemas Internet of Things - IoT) na realização das etapas de elicitação e análise de requisitos e modelagem de software.

Desenvolvedores distribuídos geograficamente podem trabalhar juntos em um espaço de trabalho compartilhado, organizando os requisitos e os diagramas. Além de fornecer técnicas específicas para análise de requisitos para IoT (ReqMLSCity) e modelagem de software para IoT (SysIoTML), o sistema oferece várias funcionalidades que podem auxiliar e melhorar o processo de desenvolvimento de software.

O que é SysIoTML?

O SysIoTML é uma extensão do SysML (Systems Modeling Language), uma linguagem de propósito geral para modelagem de sistemas de aplicações de engenharia. O SysIoTML considera aspectos específicos de Sistemas IoT (comportamento e interatividade) através dos diagramas:

  • Caso de Uso
  • Sequência
  • Máquina de Estado
  • Bloco

Referência: Nascimento et al. (2023)
